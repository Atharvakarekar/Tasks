README - Student Score Analyzer  

 Overview  
The Student Score Analyzer is a simple web application that allows users to enter student names and scores for three subjects (Math, Science, and English). It calculates the average score, assigns a grade, and displays the data in a table. Users can also sort students by name or average score.  

 Features  
- Data validation (scores must be between 0-100)  
- Automatic grade calculation  
- Sorting by name and average score  
- Dynamic table updates  

 Future Improvements  
- Edit and delete functionality  
- Search students by name  
- Improved UI design  

 Conclusion  
This project provides a simple and effective way to analyze student scores dynamically using JavaScript.
